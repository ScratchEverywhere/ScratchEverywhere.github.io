# Scratch Everywhere! Website

This is the official **website** for Scratch Everywhere!.

If you're looking for Scratch Everywhere! itself, this is the wrong repository.
Use <https://github.com/ScratchEverywhere/ScratchEverywhere> instead.

## Building

To build the site, use `bun`:

```
bun run docs:build
```

You can also use `bun run docs:dev` to run a local copy of the site for
previewing changes.
